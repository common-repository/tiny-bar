WordPress Notification Bar Plugin - TinyBar displays a notification bar to your website. You can place it to the top or bottom of your webpage. It overlaps and hides when you scroll.

== Description ==

**WordPress Notification Bar Plugin** - [TinyBar](https://tinybarwp.com/) display a notification bar to your WordPress website. You can display amazing discount offers and create urgency among your site visitors and encourage them to take up your sales offers. 

With the help of [Tiny Bar](https://tinybarwp.com/), you can easily design a fascinating Notification Bar on your WordPress website to keep everyone engaged.

You can place it top or bottom of your webpage. It overlaps and hides when you scroll. Tiny Bar allows you to display a message to your visitors with a custom button.

== Installation ==

Install TinyBar as a regular WordPress plugin. Here are different ways to install this plugin:

**Install by search plugin:**

1. In your Admin, go to menu Plugins > Add New
2. Search TinyBar
3. Click to install
4. Activate the plugin

**Install by upload plugin:**

1. Download the latest version of TinyBar (.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded tiny-bar.zip and click on "Install Now" button
6. Activate the plugin
